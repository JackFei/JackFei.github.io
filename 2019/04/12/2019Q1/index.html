2019年第一季度工作总结
Post：2019-04-12 21:39:29 Tags：/ 总结 / Visit：

一、工作内容

  1. 完成了社区8.5、8.6版本，以及对社区出现的bug及时修复。
  2. 8.5版本主要包括：专题h5、后台专题管理、小程序专题、hybrid页面专题等。
  3. 8.6版本主要包括：视频详情hybrid、bbs视频上传、hybrid视频上传优化、公开课专题等。
  4. 小程序海报、bbs漏洞修复、粉丝通知、3g帖子优化等。

二、工作收获

  1. 经过几个版本的迭代，对项目中的业务有了更深刻、全面的认识，提出的需求、问题能够快速定位并解决掉。
  2. node koa开发后端接口，以及部分中间件的使用，koa源码的阅读。
  3. 开始使用奇妙清单安排自己的时间，不过很多难以预料的事情还是不能合理安排，不断调整争取能安排好每一分钟。
  4. 学习了AST语法树相关知识，以及AST语法树在babel、postcss、eslint插件开发中的使用。
  5. git hooks、gitlab CI、github开放api、油猴脚本开发等等杂七杂八的东西。

三、需要改进的地方

  1. 年初计划执行不到位，Q2要认真落实一下，虽然学什么东西并不一定，但重要的方面还是要执行下去。
  2. 感觉有些问题考虑不到点子上，明白之后恍然大悟，要转变一下思维模式，能从结果推导原因，也能从原因推导结果，而不是毫无头绪之类的。
  3. 更加严格的执行工作流程，更好的跟同事配合。
  4. 发现项目中的问题，并能不断改进，不能因为是别人遗留的问题就得过且过，一定要不断改进才行。

四、Q2学习计划

  1. 继续学习node开发服务端、客户端脚本，出一个能够使用的作品。
  2. 对于RN可能在TOD项目落地，做好RN相关技术栈的学习。
  3. 争取写5篇以上博客，汇总、梳理学到的知识点。
  4. 积极落实一些年初的计划。
